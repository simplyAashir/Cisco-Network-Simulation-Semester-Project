#  Cisco Packet Tracer Project

A complex enterprise network simulated in Cisco Packet Tracer as part of the Computer Networks course (CL-3001), Fall 2025.

## Protocols & Features
- **Routing:** OSPF (Area 1 & 2), EIGRP (AS 11), RIP
- **Redistribution:** Between OSPF, EIGRP, and RIP on Routers 1–4
- **Subnetting:** VLSM with full subnetting tree
- **DHCP:** Dual DHCP servers for dynamic IP assignment
- **NAT:** Implemented on Router 10
- **ACL:** Access restrictions on Networks A, E, and L
